# BaseMvpProbject
本着方便快捷的原则，为了快速开发而建立，本项目是以MvpArms为雏形进行改革的，主要应用开发

#### Functionality & Libraries
1. [`Mvp` Google's official` Mvp` architecture project, which contains several different schema branches (this is the Dagger branch).](https://github.com/googlesamples/android-architecture/tree/todo-mvp-dagger/)
2. [`RxJava`](https://github.com/ReactiveX/RxJava)
3. [`RxAndroid`](https://github.com/ReactiveX/RxAndroid)
4. [`Rxlifecycle`](https://github.com/trello/RxLifecycle)
5. [`RxCache`](https://github.com/VictorAlbertos/RxCache)
6. [`RxPermissions`](https://github.com/tbruyelle/RxPermissions)
7. [`RxErroHandler`](https://github.com/JessYanCoding/RxErrorHandler)
8. [`Retrofit`](https://github.com/square/retrofit)
9. [`Okhttp`](https://github.com/square/okhttp)
10. [`Gson`](https://github.com/google/gson)
11. [`Butterknife`](https://github.com/JakeWharton/butterknife)
12. [`EventBus`](https://github.com/greenrobot/EventBus)
13. [`Glide`](https://github.com/bumptech/glide)
14. [`LeakCanary`](https://github.com/square/leakcanary)
15. [`greenDAO`](https://github.com/greenrobot/greenDAO)
16. [`ARouter`](https://github.com/alibaba/ARouter)
